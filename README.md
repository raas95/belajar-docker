## How to run

#### Windows

```bash
set PORT=8083
set INSTANCE_ID=some-text
go run main.go
```

#### Nix/Linux

```bash
export PORT=8083
export INSTANCE_ID=some-text
go run main.go
```

#### Go CLI

```bash
PORT=8083 INSTANCE_ID=some-text go run main.go
```
